# API Fitness Club

Acest folder este rezervat pentru backend-ul aplicatiei.

## Module planificate

- autentificare si autorizare;
- management utilizatori si roluri;
- management abonamente;
- clase si rezervari;
- plati si facturi;
- rapoarte pentru administrator.

## Exemple de endpoint-uri

- `POST /api/auth/login`
- `GET /api/members`
- `GET /api/classes`
- `POST /api/bookings`
- `GET /api/payments`
- `GET /api/admin/dashboard`
